# Model Ekonomi

## Fee Structure
1. **Origination Fee**: 0.5% dari jumlah pinjaman
2. **Interest Rate**: Variabel berdasarkan:
   - Jenis NFT collateral
   - Loan-to-Value ratio
   - Durasi pinjaman

## Tokenomics
- 80% fee masuk ke lender
- 15% ke treasury protokol
- 5% sebagai insentif likuidator
